Json Object & Json Array

Json Object

Đối tượng JSON được viết bên trong dấu ngoặc nhọn { }. Các đối tượng JSON có thể chứa nhiều cặp khóa: giá trị

Ví dụ:

// JSON object

{ "name": "John", "age": 22 }

Json Array

Mảng JSON được viết bên trong dấu ngoặc vuông [ ].

Ví dụ:

// JSON array

[ "apple", "mango", "banana"]


// JSON array containing objects

[

{ "name": "John", "age": 22 },

{ "name": "Peter", "age": 20 }.

{ "name": "Mark", "age": 23 }

]
